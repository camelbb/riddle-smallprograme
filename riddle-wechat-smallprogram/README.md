# 儿童谜语猜猜猜微信小程序

这是一个专为儿童设计的谜语猜猜猜微信小程序，用户可以选择不同主题的谜语进行猜答，提供简洁直观的界面和流畅的用户体验。

## 功能特点

- **主题选择**：提供多种预设主题（动物、植物、水果、蔬菜、日常用品、交通工具）
- **在线谜语获取**：通过API接口获取随机谜语，包含题目和答案
- **答案验证**：自动判断答案是否正确，并给出相应提示
- **题目切换**：支持上下一题的切换
- **得分记录**：记录用户的得分并累加
- **分享功能**：支持分享到微信好友和朋友圈
